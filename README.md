## Install dependencies

```
npm install
```

## Start server

```
npm start
```

## Deploy with pm2

Use pm2 to deploy app on production enviroment.

```
pm2 startOrReload pm2.json
```

## 前端js打包

```
生产环境 npm run fe-release
开发环境 npm run fe-dev
```

## 便捷更新生产环境

```
npm run build
```
